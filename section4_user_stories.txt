Розділ 4. Функціональні сценарії (User Stories)

Функціональні сценарії описують взаємодію користувачів із системою відповідно до їхніх потреб.
Формат User Story: Як <користувач>, я хочу <дія>, щоб <результат>.

Основні User Stories:
• адміністратор керує обліковими записами та ролями користувачів;
• оператор додає товари та фіксує надходження й відвантаження;
• оператор переглядає поточні залишки;
• керівник переглядає аналітичні звіти для прийняття рішень.

User Stories визначають базову логіку роботи та ключовий функціонал інформаційної системи.
